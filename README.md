# 🎙️ OmniCast

**Turn Any Content into an Audio Notebook.**

OmniCast is a premium, voice-first AI platform that transforms everyday content—YouTube videos, PDFs, Web Articles, and text stories—into engaging, studio-quality podcasts instantly. Powered by Murf Falcon, OmniCast features a stunning, state-of-the-art dark glassmorphism UI inspired by modern audio applications like Spotify.

## ✨ Features

- **Omni-Channel Imports:** Paste YouTube URLs, upload PDF documents, provide Web Articles, or write custom AI stories.
- **Studio-Quality Audio:** Leveraging advanced AI for incredibly realistic and professional voice synthesis.
- **Customizable Playback:** Select between different styles (Single-Speaker Educational, Conversational), granular language options, and diverse host voices.
- **Premium User Interface:** An edge-to-edge dark mode application customized with sleek Tailwind CSS, featuring immersive ambient background glows and borderless translucent glass cards.

## 🚀 Getting Started

First, install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🛠️ Tech Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS v4
- **Components:** Custom Built Glassmorphism UI
- **Deployment:** Vercel
